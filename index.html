    • Tugas :
      • Buktikan jika L = (x-a)^2 . (y-b)^2 = r^2 maka
        PGS = (x-a) . (x1-a) + (y-b) . (y1-b) = r^2
      • Buktikan jika L = x^2 + y^2 + Ax + By + C = 0 maka
        PGS = x.x1 + y.y1 + A((x + x1)/2) + B((y + y1)/2) + C = 0
